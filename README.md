Iris Flower Classification is a machine learning project that predicts the species of an iris flower based on four features:
- *Sepal Length*
- *Sepal Width*
- *Petal Length*
- *Petal Width*

The model classifies flowers into one of three species:
1. *Iris Setosa*
2. *Iris Versicolor*
3. *Iris Virginica*
